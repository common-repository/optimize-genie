== Description ==

"OptimizeGenie" is a WordPress plugin that helps you optimize your website to get the best performance and speed. This plugin is very easy to use and has a very simple user interface.

With "Optimize Genie" you can optimize your website in a few clicks.

This plugin will help you:

- Manipulate and Customize the sizes on your images.
- Add ?width={number} to the end of your image URL to resize it on the fly. This includes all images under wp-content/, which inlcudes themes, plugins or media library images.

== Installation ==

Upload the optimize-genie folder to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
Go to the Optimize Genie settings page to configure your optimization's settings.
